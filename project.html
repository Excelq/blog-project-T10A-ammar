Shark Attack

Shark Attack is a very simple game and is the first scratch game i made. The visuals are cartoony, but the assets are high quality and sharp. It uses an infinitely repeated blocks of code to clone the prey fish over and over again with a different costume each time the clone is summoned. The shark moves follows your cursor, and the object of the game is to collect as much point as possible. You can even change the costume sprites if you want, as the code is not very complex, and you could figure out what blocks of code does what after a few minutes glancing and exploring the code.

Buditale

Buditale is a game i made for the final project. It attempts to copy the battle system of the game Undertale, except the assets are a bit low quality and weird because i had to make them myself. There are a lot of bugs in the codes as i didnt have time to double check the code and actually polish the game. Some attacks work, and the mechanics are no different from Undertale, you can fight, or act, look at your items, and spare your opponent, in this case, Budi. It has no actual story, and it was only intended to copy the battle system of Undertale.
